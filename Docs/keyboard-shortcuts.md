# Keyboard Shortcuts

## Playback

| Key | Action |
|-----|--------|
| Space | Play / Pause |
| J | Reverse (0.5×) |
| K | Pause |
| L | Fast-forward (1.5×) |
| ← | Seek back 5 seconds |
| → | Seek forward 5 seconds |

## Editing

| Key | Action |
|-----|--------|
| Delete / Backspace | Delete selected words |
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |

## App

| Key | Action |
|-----|--------|
| Ctrl+S | Save project |
| Ctrl+E | Export |
| ? | Show shortcut cheatsheet |
